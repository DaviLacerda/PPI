Explicação

A saída produzida pelo código é igual a:

        [object Text]
        [object HTMLHeadingElement]
        [object Text]
        [object HTMLInputElement]
        [object Text]
        [object Comment]
        [object Text]
        [object HTMLPreElement]
        [object Text]
        [object HTMLScriptElement]
        true
        true
        true
        true
        true
        [object HTMLDocument]

A saída apresentada se deve ao fato de como a DOM Tree é construída. Os elementos h1, input, pre são nós que herdam propriedades anteriores de objetos HTMLElement, Element, Node e EventTarget, isto explica o resultado das linhas 11 até 15. Além disso, qualquer texto que não esteja dentro desses nós é representado por um nó do objeto Text, sendo na maioria das vezes espaços em branco/tabulações criadas para formatação do código pelo navegador (Devtools -> Fontes). Por fim, o elemento document é criado herdando propriedas de outros elementos anteriores, assim como mencionado acima - com exceção das propriedades de objetos HTMLElement e Element.
